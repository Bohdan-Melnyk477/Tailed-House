# 📌 Про проєкт

**Хатинка Лапок** — це вебзастосунок для притулку тварин, який допомагає людям знайти нового друга серед безпритульних тварин.

Сайт дозволяє користувачам:

- переглядати список тварин
- фільтрувати їх за категоріями
- переглядати детальну інформацію про кожну тварину
- залишати заявку на знайомство або усиновлення

🎯 **Мета проєкту** — допомогти притулкам швидше знаходити нові домівки для тварин.

# 🧩 Основні секції сайту

## Header
Містить:
- логотип
- навігацію
- кнопку **"Взяти друга"**

---

## Hero Section

Головний екран сайту, який містить:

- головний заголовок
- текстовий опис
- кнопку переходу до історій
- фонове зображення

---

## Pets List Section

Секція зі списком тварин.

Картка тварини містить:

- зображення
- кличку
- вид
- вік
- стать
- короткий опис
- кнопку **"Дізнатись більше"**

---

## About Us Section

Секція **"Чому Хатинка Лапок"**

Містить:

- заголовок
- слайдер із текстом та зображеннями
- кнопки навігації

---

## Success Stories Section

Секція з відгуками користувачів.

Містить:

- список відгуків
- рейтинг зірками
- ім’я користувача
- слайдер

---

## Animal Details Modal

Модальне вікно з детальною інформацією про тварину.

Містить:

- зображення тварини
- кличку
- вид
- вік
- стать
- опис
- інформацію про здоров’я
- кнопку **"Взяти додому"**

---

## Order Modal

Модальне вікно з формою заявки.

Форма містить:

- ім’я
- телефон
- коментар

Після відправки заявки з’являється **push-повідомлення**.

---

## Footer

Футер містить:

- логотип проєкту
- блок навігації по сторінці
- іконки соціальних мереж
- авторські права

Соціальні мережі:

- Instagram
- Facebook
